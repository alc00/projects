Project Details:
The goal of the project is to achieve the following:
1) Using user-inputted information, project the user's income, expenses, and net worth through the user's lifetime.
2) Predict the age of when the client's net worth will turn into negative (i.e. exhaustion of the user's retirement savings).
3) Show the economic activity generated by the user from now to the end of the user's lifetime.


Tools/Languages Used:
1) R
